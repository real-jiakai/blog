本站会自动记录访问，用于了解读者、提高可靠性和改进内容，不提供分析工具的同意弹窗。根据具体服务，记录的信息可能包括访问页面与来源页面、访问时间、交互或会话回放、浏览器与设备信息、由 IP 推断的大致位置，以及技术诊断数据。

正式站点使用以下服务：

- [Microsoft Clarity](https://privacy.microsoft.com/privacystatement)：交互分析与会话回放。
- [Google Analytics](https://policies.google.com/privacy)：流量与受众分析。
- 自托管的 [Umami](https://umami.is/privacy)：流量分析。
- 由站点交付基础设施提供的 [Cloudflare Web Analytics](https://www.cloudflare.com/privacypolicy/)。

文章中出现相关功能时，表情反应和评论分别由 emaction 与 GitHub Utterances 处理。访问外部链接后，该网站自己的隐私规则将适用。

浏览器隐私设置、内容拦截器或网络策略可能阻止部分请求。如对本站的数据处理有疑问，可通过[关于页面](/about/)中的联系方式联系我。
